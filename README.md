# E-commerce Product Page

A modern, responsive e-commerce product page with cart functionality and image gallery.

![E-commerce Screenshot](./screenshot.png)

**🔗 Live Demo:** [View Live Demo](https://kevinwangombe.github.io/ecommerce-product-page)

## ✨ Features

- Product image gallery with thumbnails
- Add to cart functionality
- Cart counter with animations
- Quantity selector
- Responsive design (mobile, tablet, desktop)
- Smooth transitions and hover effects

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6+)
- Local Storage (cart persistence)

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/kevinwangombe/ecommerce-product-page

# Open index.html in browser
# Or use a local server:
npx serve
```
